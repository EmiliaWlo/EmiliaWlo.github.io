Selenium Demo with NodeJS

November 2022

In this project, I used Selenium in a NodeJS environment to smoke-test a couple of common user pathways for Barclays. Because of EU regulations, the automation required accepting the cookies. The project additionally demos running simultaneous chrome driver instances to run test in parallel, as well as several common selenium testing tchniques, such as xpaths selectors, wait statements and page interaction.
